## Overview

This is an **educational simulator of *ab initio* Molecular Dynamics** using the Hartree–Fock method with the STO-3G basis set.

The goal is **not** high performance or top-level scientific accuracy, but to provide an **interactive environment** where students can visualize and understand key concepts of computational quantum chemistry in real time.

These include:

- Molecular orbitals  
- Population analysis  
- Energy decomposition  
- Forces  
- Real-time dynamics  

---

## Purpose

This project is designed for **learning and teaching**, helping students move beyond static theory and interact directly with:

- Electronic structure
- Forces and motion
- Energy contributions during simulation
