> This is an educational simulator of ab initio Molecular Dynamics using Hartree–Fock with the STO-3G basis set.
The goal is not high performance or top-level scientific accuracy, but rather to allow students to visualize and understand the core concepts of computational quantum chemistry in an interactive way: molecular orbitals, population analysis, energy decomposition, forces, and real-time dynamics.
